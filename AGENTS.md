# AGENTS.md

## 프로젝트 개요

- Next.js 16, React 19, TypeScript strict 기반 웹 애플리케이션입니다.
- 루트 설정 파일을 우선 확인하고, 필요한 경우에만 관련 소스 파일을 제한적으로 읽습니다.

## AI 에이전트 작업 원칙

에이전트는 작업을 시작하기 전에 아래 항목을 간략히 정리해 사용자에게 확인받습니다.

1. 요청 해석: 사용자의 목표와 변경 범위를 한두 문장으로 요약합니다.
2. 영향 범위: 수정이 필요한 파일 또는 모듈 후보를 제시합니다.
3. 실행 단계: 조사, 구현, 검증 순서로 작업 목록을 제시합니다.
4. 확인 요청: 구현에 들어가기 전에 사용자에게 진행 여부를 확인합니다.

## 단계별 진행 방식

- 1단계: 루트 설정 파일과 직접 관련된 파일만 확인합니다.
- 2단계: 필요한 변경 범위를 사용자에게 설명하고 확인받습니다.
- 3단계: 확인된 범위 안에서만 코드를 수정합니다.
- 4단계: 가능한 경우 타입 검사, 빌드, 린트 등 최소 검증을 수행합니다.
- 5단계: 변경 내용, 검증 결과, 남은 리스크를 간략히 보고합니다.

## 제한 사항

- 전체 디렉터리를 불필요하게 스캔하지 않습니다.
- 사용자 확인 없이 대규모 리팩터링이나 구조 변경을 진행하지 않습니다.
- 기존 사용자 변경사항을 되돌리지 않습니다.
