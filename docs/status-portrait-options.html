상태창에 캐릭터 이미지 넣기 — 4가지 방법

방법 3

이미지 블리드 (배경 합성)

높이 변화 없음

칩 오른쪽에 일러스트를 깔고 왼쪽으로 페이드(마스크)시켜 텍스트와 겹치게 합니다. 높이는 그대로 두면서 카드 같은 분위기. 화려하지만 텍스트 가독성에 신경 써야 합니다.

+ 높이 유지, 분위기 ↑− 텍스트/이미지 대비 관리 필요
