# 던전 TRPG — 게임 설계 문서

> AI 나레이터(LLM)와 함께 진행하는 텍스트 기반 던전 탐험 TRPG.
> 플레이어는 전사를 조작하며, AI가 GM 역할로 동료(피나·미나) 및 환경 묘사를 담당한다.

---

## 1. 개요

### 1.1 게임 컨셉
- **톤: 클래식 왕도 JRPG 풍** 중세 판타지. 플레이어가 콘솔 RPG의 주인공이 된 듯한 몰입감을 지향.
- 3층 구조의 미궁형 던전을 탐험하며 각 층의 보스 몬스터를 격파, 최하층 레드드래곤 처치로 클리어.
- 모든 주요 행동은 **1d36 다이스 판정**으로 결과 결정.
- 패배는 게임의 일부로 설계되어 있어 부정적 종결이 아닌 긴장감을 위한 장치.

### 1.2 핵심 게임 루프
1. 방 진입 → 방 타입에 따른 이벤트 발생
2. AI 나레이션 → 플레이어가 선택지 선택
3. 다이스 판정 → 결과에 따른 상태 갱신 (HP·인벤·영감·위치 등)
4. 인접 방으로 이동하거나 행동 반복
5. 보스 격파 시 다음 층 이동, 최종 보스 격파 시 정복 엔딩

### 1.3 등장 캐릭터
| 분류 | 이름 | 클래스 | 성격 / 외형 |
|---|---|---|---|
| 플레이어 | (사용자 지정) | 전사 | — |
| 동료 | 피나 | 도적 | 수다스럽고 활기찬 분위기메이커 (ENFP 7w6, 오렌지빛 단발·푸른 눈) |
| 동료 | 미나 | 마법사 | 이지적이고 침착함 (INTJ 5w6, 긴 은발·푸른 눈) |
| NPC | 린린 | 마법사 | 소심하고 예의바름 (ISFJ 9w1, 보랏빛 장발·보랏빛 눈동자, 마녀 모자·로브) |

## 2. 스탯 시스템

### 2.1 기본 스탯
- 💪 **근력 (STR)**: 힘으로 해결하는 행동 (철문 개방, 낙석 저지 등)
- 🏃 **민첩 (DEX)**: 빠르고 정밀한 행동 (질주, 도약, 회피 등)
- 💡 **지능 (INT)**: 지적 행동 (마법 해독, 환영 간파 등)

### 2.2 초기 캐릭터 상태
| 캐릭터 | HP | 💪 | 🏃 | 💡 | 특수액션 |
|---|---|---|---|---|---|
| 전사 (플레이어) | 10/10 | 4 | 1 | 0 | 강타(3/3), 활로개척(3/3) |
| 피나 (도적) | 10/10 | 1 | 4 | 1 | 암습(3/3), 패스파인딩(3/3) |
| 미나 (마법사) | 10/10 | 0 | 1 | 4 | 마력속박(3/3), 연금생성(3/3) |

### 2.3 초기 장비
| 캐릭터 | ⛑️ 머리 | 🧥 몸통 | 🗡️ 무기 |
|---|---|---|---|
| 전사 | 없음 | 낡은 철갑옷 | 낡은 철검 |
| 피나 | 없음 | 낡은 천갑옷 | 낡은 대거 |
| 미나 | 없음 | 낡은 로브 | 낡은 스태프 |

### 2.4 초기 파티 상태
- 소지품: 회복물약 ×2
- 현재 층: 1️⃣
- 현재 위치: 던전 입구 (C6)
- 영감: ★☆☆ (시작 시 1)

## 3. 다이스 및 판정 시스템

### 3.1 다이스
- 모든 판정은 **1d36** (1~36 균등난수) 1회 굴림.
- 굴린 값은 절대 임의 변경 불가 (긍정화·재굴림 금지).

### 3.2 판정 공식
- **정석적 선택**: `최종값 = 1d36 + 해당 스탯 + 영감 보정`
- **비정석적 선택**: `최종값 = 1d36` (스탯 미합산)
- 영감 보정: 사용 시 +5, 미사용 시 0

### 3.3 난이도 모드
플레이어 선언으로 변경 가능 (기본 = 노말).

| 모드 | 대성공 | 성공 | 실패 | 대실패 |
|---|---|---|---|---|
| 🔥 하드 | 42 이상 | 30–41 | 16–29 | 1–15 |
| 📜 노말 (기본) | 38 이상 | 25–37 | 11–24 | 1–10 |
| 😎 이지 | 30 이상 | 18–29 | 10–17 | 1–9 |

### 3.4 비정석적 선택의 특칙
- 모드와 무관하게 다이스 26 이상이면 성공, 이하는 실패.
- 성공 시: 행동을 수행한 캐릭터의 특수액션 1회 회복 (3 초과 불가).

## 4. 영감 시스템

### 4.1 보유
- 별 단위 표기: ★ = 보유, ☆ = 없음.
- 시작 시 ★☆☆ (1개).
- 최대 ★★★ (3개). 4개 이상은 무효 처리.

### 4.2 획득
- 👹 몬스터방 전투 승리 직후에만 +1.
- 단순 이동, 함정 돌파, 수수께끼 성공으로는 회복되지 않음.

### 4.3 사용
- 발동 선언 시 다음 판정에 +5 보정.
- 발동 시 즉시 ★ 1개 차감.

### 4.4 검증 원칙
- 영감 발동 선언 시 엔진은 반드시 현재 수치를 확인.
- 0(☆☆☆) 상태에서 발동 요청 시 무시하고 일반 판정 진행.

| 👹 | 몬스터방 | 몬스터 출현, 격파 시 영감 1 회복 |
| Ⅰ / Ⅱ / Ⅲ | 몬스터 난이도 | 1~3단계 (Ⅲ이 최강) |
| 🗝️ | 수수께끼의 방 | 난관 발생 (돌파 또는 후퇴) |

### 5.6 계층 돌파 보상
보스 격파 후 계단 사용 시:
- 다음 층으로 이동
- 아군 전원 HP 완전 회복
- 전원 특수액션 1회 회복 (3 초과 불가)
- 추가로 전설(🐦‍🔥) 아이템 획득:
  - 리치 격파 → 1개 / 발록 격파 → 2개 / 레드드래곤 격파 → 게임 클리어

### 5.7 수수께끼의 방 (🗝️)
- 대표자 1인이 판정 (비정석적 선택 가능). 결과는 파티 전체 적용.
- 난관 카테고리 예시:
  - 💪: 거대 철문 개방 / 낙석 저지 / 오크와 팔씨름
  - 🏃: 독침 복도 질주 / 무너지는 발판 선별 / 도약과 절벽 통과
  - 💡: 마법문 해독 / 마력 흐름 역전 / 환영 간파
- 결과:
  - 대성공 → 희귀 장비 1개 + 돌파
  - 성공 → 일반 장비 1개 + 돌파
  - 실패/대실패 → 전체 HP −1 + 재도전 가능

### 5.8 함정의 방 (🕸️)
- 다이스 체크로 돌파 또는 후퇴 선택.
- 함정 예시: 바닥 가시 덫 / 벽면 화살 / 천장 낙하 그물 / 마력 폭발 / 수면 가스
- 결과:
  - 대성공 → 무손실 돌파 + 성공자의 특수액션 전부 1회 회복 (활로개척 사용으로 인한 대성공 포함)
  - 성공 → 무손실 돌파
  - 실패/대실패 → HP 1 소모 + 돌파

### 5.9 NPC 린린 (🧙‍♂️, 2F의 E6)
- 응원 문구 전달 후 다이스 체크 1회.
  - 대성공 → 희귀 장비
  - 성공 → 일반 장비
  - 실패/대실패 → 회복물약
- 보상 전달 후 즉시 퇴장 (재방문 시 사용 불가).

## 6. 전투 시스템

### 6.1 전투 진행 원칙
- 몬스터 HP가 정확히 0 이하가 될 때까지 전투 종료 불가.
- 전투 중 가능한 선택지: **공격 / 특수액션 / 도주**.
- 플레이어는 매 턴 **공격을 주도할 캐릭터를 선택**하여 다이스 굴림 진행.
- 공격자 1회 ↔ 몬스터 1회 **교차 피격**.
- 공격자의 HP 차감 = 몬스터 데미지 (다이스 판정 결과와 무관, 공격 시 고정 발생).
  - **예외 (몬스터 데미지 미발생)**: 대성공 / 암습 사용 / 속박 상태

### 6.2 공격 판정 결과
| 결과 | 효과 |
|---|---|
| 대성공 | 몬스터 HP −5, 몬스터 공격 회피 |
| 성공 | 몬스터 HP −2 |
| 실패 | 몬스터 HP 변화 없음 |
| 대실패 | 몬스터 HP 변화 없음, 공격자 장비 일시 유실 (재장착에 1턴 소요) |

### 6.3 일반 몬스터 데미지
- 데미지 = 난이도 수치 (Ⅰ=1, Ⅱ=2, Ⅲ=3)

### 6.4 보스 몬스터
| 층 | 보스 | HP | 기본 데미지 | 특수공격 |
|---|---|---|---|---|
| 1F | 리치 | 25 | 전체 1 (혹한마법, 매턴 발동) | — |
| 2F | 발록 | 28 | 2 | 악마의 저주 — 2턴에 1회, 무작위 1인의 장비를 이번 전투 동안 무력화 (전투 종료 시 복구) |
| 3F | 레드드래곤 | 30 | 3 | 화염 브레스 — 3턴에 1회 준비 → 다음 턴 전체 데미지 5 발사. 준비 중 공격 성공·대성공 시 1턴 기절 + 브레스 무효화 |

### 6.5 도주
- 다이스 체크 → 성공 이상이면 직전 방으로 후퇴.
- 실패 시 몬스터에게 1회 피격 후 전투 지속.
- 연막탄 사용 시 판정 없이 후퇴 확정.

### 6.6 패배 조건
- 파티 전원 HP 0(의식불명) 시 패배.
- 패배 시 메타 엔딩 출력 (§10.2 참조).

### 6.7 상태 이상 표기
- 마력속박 적용: 몬스터 HP 아래 `현재상태: 속박(2턴 남음)` 표기
- 레드드래곤 기절: `현재상태: 기절(1턴 남음)` 표기

## 7. 특수액션

### 7.1 사용 한도
- 캐릭터당 각 액션 3회까지 사용. **4회 이상 회복 불가** (오버플로우 금지).

### 7.2 액션 목록
| 액션 | 분류 | 캐릭터 | 효과 |
|---|---|---|---|
| 강타 | 전투 | 전사 | 다음 공격 무조건 대성공 |
| 활로개척 | 탐험 | 전사 | 수수께끼/함정의 방 무조건 대성공 |
| 암습 | 전투 | 피나 | 몬스터 공격 회피 + 자신의 공격은 다이스 판정 |
| 패스파인딩 | 탐험 | 피나 | 보스룸의 방향 지시 |
| 마력속박 | 전투 | 미나 | 몬스터 2턴 행동불능 |
| 연금생성 | 탐험 | 미나 | 다이스 판정 → 대실패/실패 = 회복물약, 성공 = 상급회복물약, 대성공 = 엘릭서 |

### 7.3 회복 조건
다음의 경우 각 1회 회복 (3 초과 불가):
- 계층 돌파 (계단 사용 시 전원 회복)
- 함정의 방 대성공
- 비정석적 선택 성공 (성공한 본인만)

## 8. 아이템 및 장비

### 8.1 부위 슬롯
- ⛑️ 머리 / 🧥 몸통 / 🗡️ 무기 / 🧪 소모품
- 머리·몸통·무기는 각 1개씩만 장착 가능. **새 장비 장착 시 같은 부위 기존 아이템은 즉시 파기**.

### 8.2 일반 등급 📦 (스탯 +1)
보물의 방 다이스 판정 **대실패~실패** 시 획득.

| 부위 | 💪 | 🏃 | 💡 |
|---|---|---|---|
| ⛑️ 머리 | 철제 투구 | 도적의 복면 | 현자의 서클릿 |
| 🗡️ 무기 | 강철 대검 | 예리한 단검 | 참나무 지팡이 |
| 🧥 몸통 | 사슬 갑옷 | 가죽 슈트 | 마법사 로브 |

**소모품**: 회복물약 (HP +3), 연막탄 (전투 확정 회피=후퇴)

### 8.3 희귀 등급 🎁 (스탯 +2)
보물의 방 다이스 판정 **성공~대성공** 시 획득.

| 부위 | 💪 | 🏃 | 💡 |
|---|---|---|---|
| ⛑️ 머리 | 미노타우르스의 뿔 | 바람의 머리띠 | 고대 예언자의 관 |
| 🗡️ 무기 | 미스릴 해머 | 암살자의 쌍검 | 수정 보주 |
| 🧥 몸통 | 판금 갑옷 | 고양이 발걸음 예복 | 별빛의 가운 |

**소모품**: 상급 회복물약 (HP +5), 마나물약 (특수액션 +1)

### 8.4 전설 등급 🐦‍🔥 (스탯 +3)
**오직 보스 💀 처치 시에만 획득.**

| 부위 | 💪 | 🏃 | 💡 |
|---|---|---|---|
| ⛑️ 머리 | 티탄의 왕관 | 심연의 그림자 두건 | 리치의 영혼 왕관 |
| 🗡️ 무기 | 드래곤 슬레이어 | 신속의 칼날 '엑스큐션' | 신의 지혜 '아카샤' |
| 🧥 몸통 | 아다만티움 흉갑 | 신기루의 외투 | 대마법사의 성의 |

**소모품**: 엘릭서 (HP +10), 성녀의 눈물 (전원 HP +5)

### 8.5 장비 교체 규칙
- 새 장비 습득 시 반드시 **[기존 보너스 제거] → [새 보너스 적용]** 순서로 처리.
- 표기는 `기본 + 보너스` 형식.
- **올바른 예시**: 강철 대검(💪+1) 장착 중 드래곤 슬레이어(💪+3)로 교체
  → 강철 대검 +1 제거 → 드래곤 슬레이어 +3 적용 → 최종 💪 `4+3`
- **잘못된 예시**: 보너스 누적 표기 (`4+1+3`) — 금지

### 8.6 데이터 동기화 원칙 (엔진 구현 필수)
- 아이템 획득 → 즉시 `party.inventory`에 추가
- 장비 장착 → `inventory`에서 제거 + 해당 캐릭터 `equip` 갱신 + 능력치 합산
- 상태 표시 시 변하지 않은 데이터도 **항상 전체 객체 전송** (부분 전송 금지, 데이터 손실 위험)

## 10. 엔딩

### 10.1 정복 엔딩
- 레드드래곤 격파 시 발동.
- 시점이 TRPG 테이블로 전환. 피나와 미나가 환호성을 지르고, 간식을 먹으며 오늘 세션의 하이라이트를 떠드는 장면으로 마무리.
- 정복 엔딩 이미지 출력.

### 10.2 패배 엔딩
- 파티 전원 HP 0 도달 시 발동.
- 시점이 TRPG 테이블로 전환. 세션을 마친 피나·미나가 후기를 나누며 훈훈하게 마무리.
- 패배 엔딩 이미지 출력.

> 패배는 부정적 종결이 아닌 TRPG 경험의 일부. 긴장감을 위한 장치로 설계됨.

### 11.4 UI 컴포넌트 스펙

**선택지 컴포넌트 `<TRPGChoice />`**
- Props:
  - `choice1`~`choice3`: 버튼 라벨 (짧은 행동 키워드, 예: `"🧭 북쪽으로 이동"`)
  - `text1`~`text3`: 클릭 시 LLM에 전송될 시도 묘사문 (번호·머리말 없음)
- 위치: 나레이션과 상태창 사이.

**상태창 컴포넌트 `<StatusWindow />`**

### 11.5 게임 상태 모델 (서버/클라이언트 공통)
```ts
type ItemRarity = "common" | "rare" | "legendary";
type Slot = "head" | "body" | "weapon" | "consumable";

interface Item {
  id: string;
  name: string;
  rarity: ItemRarity;
  slot: Slot;
  bonus?: { str?: number; dex?: number; int?: number };
  effect?: string;   // 소모품용
}

interface Character {
  name: string;
  hp: number;
  maxHp: number;
  baseStats: { str: number; dex: number; int: number };
  equip: { head: Item | null; body: Item | null; weapon: Item | null };
  actions: { name: string; remaining: number; max: number }[];
}

interface Party {
  members: [Character, Character, Character]; // [전사, 피나, 미나]
  inventory: Item[];
  floor: 1 | 2 | 3;
  position: { col: string; row: number };     // 예: { col: "C", row: 6 }
  inspiration: 0 | 1 | 2 | 3;
}

interface CombatState {
  monster: MonsterRuntimeState | null;
  bossCounters?: {
    curseTurn?: number;     // 발록
    breathTurn?: number;    // 레드드래곤
  };
  monsterStatus?: { type: "속박" | "기절"; remaining: number };
}

interface GameState {
  party: Party;
  combat: CombatState;
  mode: "easy" | "normal" | "hard";
  log: string[];
}
```

## 부록 A. 디자인 노트

- **다이스 산수**: 1d36 + 스탯(최대 4) + 영감(+5) = 이론상 최대 45.
  - 하드모드 대성공(42 이상)은 영감 + 높은 스탯이 모두 필요해 후반 보상감을 키움.
  - 노말모드 평균(중앙값) ≈ 18 + 스탯 → 성공권 진입 부담 적당.
- **밸런스 안전장치**: 보스의 전체 데미지(리치=1, 발록 저주, 레드드래곤 브레스=5)는 안전지대(⛺)·상급 회복물약·전설 소모품(성녀의 눈물=전원+5)으로 상쇄 가능한 분포.
- **영감 절약 압박**: 영감은 오직 👹 격파로만 회복되므로, 함정·수수께끼에 영감을 쓰면 보스전에서 부족해질 위험 → 자원 관리 긴장감 유도.
- **비정석적 선택의 의도**: 스탯 미합산 페널티로 리스크는 크지만, 성공 시 특수액션 회복으로 보상 → 창의적 플레이 유도.
